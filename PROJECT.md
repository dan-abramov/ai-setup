# Описание

Проект генерирует ruby-задачи по программированию для пользователя.
Последний указывает что ему хочется отточить. Например, "сортировка пузырьком", если он новичок.

Далее, указывает вокруг какой темы должно составляться описание задачи. Чтобы повысить свою вовлеченность в решение задачи. Например, тема: "Warhammer 40K".
В результате, как пример, будет должно получиться описание: 
"Представим боевой порядок, как массив из чисел. Где 1 - отряды Астра Милитарум, 2 - танки типа "Адская гончая", 3 - отряды Астартес, 4 - Имперские Рыцари. 
 В результате ошибок в навигации и десантировании боевые порядки перемешались и сейчас выглядят как [2,1,4,3]. Напишите алгоритм, который поможет расставить силы правильно.
 Чтобы избежать дизорганизации и лишних потерь."

После того как пользователь указал точку интереса и тему, для него генерируется описание задачи + тесты, которые проверят, что он написал правильный код.
Юзер читает описание и решает проблему. Затем его решение пропускается через тесты и он получает вердикт.
